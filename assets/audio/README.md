# Audio assets (optional)

Put audio files in this folder with these filenames:

- `ambient_hum.mp3` (or any format your browser can play)
- `door_clunk.mp3`
- `camera_blip.mp3`
- `jumpscare_stinger.mp3`

If these files are missing, the game uses simple synthesized beeps (so it remains playable out of the box).
